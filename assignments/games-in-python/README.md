
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game in Python that uses strings, loops, and user input. Practice control flow, data structures, and user interaction.

## 📝 Tasks

### 🛠️ Task 1 — Core Hangman

#### Description
Implement the core Hangman game loop: choose a secret word, accept letter guesses, reveal matched letters, and track remaining attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses from the player
- Display the current word progress using underscores for unknown letters (e.g. _ a _ _ _)
- Track and display remaining incorrect attempts
- Detect win and loss conditions and print a clear message

### 🛠️ Task 2 — Enhancements (optional)

#### Description
Add at least one enhancement to improve gameplay or user experience.

#### Examples

- Add difficulty levels that change the number of attempts
- Provide a hint system that reveals one letter
- Allow the player to save and load game progress
- Read words from an external `data.csv` or a larger word list

## 📂 Starter files

- `starter-code.py` — starter template with basic I/O and scaffolding
- `data.csv` — (optional) word list for extra-credit enhancements

## ▶️ How to run

Run the game with Python 3:

```bash
python3 starter-code.py
```

## ✅ Submission

1. Ensure your program runs without errors.
2. Add comments explaining key functions and decisions.
3. Submit your `starter-code.py` (and any added files) in the assignment folder.

## 📚 Skills practiced

- String manipulation
- Loops and conditionals
- Basic file I/O (optional)
- Program flow and user interaction

Good luck — have fun building your game!
